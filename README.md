 About the Project

- FlavourNest is a frontend restaurant web application that simulates a real food ordering experience.  
It features a modern UI, dynamic menu, and shopping cart system inspired by food delivery platforms like Swiggy and Zomato.

- This project was built to improve my frontend development and JavaScript skills.

 Features

- 20+ Hyderabad famous food items
-  Add to cart functionality
-  Live total price calculation
-  Fully responsive design 
-  Clean and modern UI
-  Interactive user experience
-  Smooth animations 

Tech Stack

- HTML5
- CSS3
- JavaScript 
- Git & GitHub
- GitHub Pages (Deployment)
